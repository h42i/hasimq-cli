hasimq-cli
==========

A simple command line MQTT client.

## Usage

```./hasimq-cli set socket (1|2|3) (on|off)```

```./hasimq-cli set strip $color```

```./hasimq-cli set mpd (play|pause)```

```./hasimq-cli get (temperature|pressure|light|random_byte)
